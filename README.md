# Smart Study Planner System 📚

A Java web application built using JSP, Servlets, MySQL and Apache Tomcat.

## Features
- User Registration & Login
- Add Study Tasks
- View Tasks
- Track Status

## Technologies
- Java (Servlets & JSP)
- MySQL
- Apache Tomcat
- HTML, CSS

## How to Run
1. Import project into Eclipse
2. Configure Tomcat server
3. Create database in MySQL
4. Update DBConnection.java
5. Run project
